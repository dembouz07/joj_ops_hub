# 📱 Installation PWA sur iOS - JOJ OPS HUB

## Guide Complet pour iPhone et iPad

### 📋 Prérequis

- iPhone ou iPad avec iOS 15.4 ou supérieur
- Safari (navigateur par défaut iOS)
- Connexion internet
- URL de votre application Vercel

---

## 🚀 Installation Étape par Étape

### Étape 1 : Ouvrir l'Application dans Safari

1. **Ouvrez Safari** sur votre iPhone/iPad
2. **Tapez l'URL Vercel** dans la barre d'adresse :
   ```
   https://joj-ops-hub.vercel.app
   ```
   (ou votre domaine personnalisé)
3. **Appuyez sur Entrée**
4. **Attendez** que la page se charge complètement

> ⚠️ **Important** : Vous DEVEZ utiliser Safari. Chrome et autres navigateurs iOS ne supportent pas l'installation PWA.

---

### Étape 2 : Ajouter à l'Écran d'Accueil

1. **Appuyez sur le bouton Partager** (□↑)
   - Il se trouve en bas de l'écran sur iPhone
   - En haut à droite sur iPad

2. **Faites défiler vers le bas** dans le menu qui s'ouvre

3. **Appuyez sur "Sur l'écran d'accueil"** (ou "Add to Home Screen")
   - Icône : un carré avec un +

4. **Personnalisez le nom** (optionnel)
   - Par défaut : "JOJ OPS HUB"
   - Vous pouvez le raccourcir en "JOJ OPS"

5. **Appuyez sur "Ajouter"** en haut à droite

---

### Étape 3 : Lancer l'Application

1. **Retournez à l'écran d'accueil**
2. **Trouvez l'icône JOJ OPS HUB** avec le logo
3. **Appuyez sur l'icône** pour lancer l'app

---

## 🔧 Dépannage iOS

### Problème : Le bouton "Sur l'écran d'accueil" n'apparaît pas

**Solutions :**
1. Vérifiez que vous utilisez **Safari** (pas Chrome)
2. Assurez-vous d'être sur **iOS 15.4+**
3. Rechargez la page (tirez vers le bas)
4. Videz le cache Safari :
   - Réglages → Safari → Effacer historique et données

### Problème : L'icône n'apparaît pas correctement

**Solutions :**
1. Supprimez l'app de l'écran d'accueil
2. Fermez Safari complètement
3. Rouvrez Safari et réessayez
4. Vérifiez votre connexion internet

### Problème : L'app ne fonctionne pas hors ligne

**Solutions :**
1. Ouvrez l'app une première fois avec internet
2. Naviguez sur plusieurs pages
3. Fermez l'app
4. Attendez 30 secondes
5. Rouvrez → devrait fonctionner hors ligne

### Problème : L'app s'ouvre dans Safari au lieu du mode plein écran

**Causes possibles :**
- Vous avez cliqué sur un lien externe
- Vous avez ouvert l'URL dans Safari au lieu de l'icône
- L'app a été mal installée

**Solution :**
- Supprimez l'app de l'écran d'accueil
- Réinstallez-la en suivant les étapes ci-dessus
